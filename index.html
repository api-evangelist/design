API Design

API Design is a project I have spun off my core API research and monitoring around API managementservice providers. It isn't a category that is full of fully defined and has a bunch of tools and services quite yet, but it needs to be. There are plenty of passionate folks about API design, we just need to expand the audience.

I consider API design, everything that goes into planning and designing your API, as well as the design of your API that lives on as your operational interface.

Your API design will be the definition of each endpoint, its methods, fields and more. In a technical sense it is a JSON or XML blueprint that describes your API, and in a creative sense, API design is an art and can possess a strange technical beauty.

Thoughtful API design early on can save you a lot of time down the road.  This work is not meant to endorse any particular approach or methodology, but provide a single resource where you can find the best information on API design.

Right now there are only a handful of services and tools available. I will be finding the best news, analysis and other resources to help you with API design. As my research evolves, I will list more, if there is anything you think so be listed, let me know.
