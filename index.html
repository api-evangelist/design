API Design

API Design is a project I spun off my research around API service proviers. It isn't a category that is full of tools and services quite yet, but it needs to be.

I'm consider API design, everything that goes into planning and designing your API, as well as the design of your API that lives on as your API.

Your API design will be the definition of each endpoint, its methods, fields and more. In a technical sense it is a JSON or XML blueprint that describes your API, and a creative way API design is an art and can possess beauty.

Right now there are a handful of services and tools available. I will be finding the best news, analysis and other resources to help you with API design. As my research evolves, I will list more, if there is anything you think so be listed, let me know.
